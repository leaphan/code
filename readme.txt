The file "data" contains 46 datasets. You can also find them in the KEEL, UCI and Libsvm public databases;
the websites are as follow:
https://sci2s.ugr.es/keel/imbalanced.php  (KEEL)
http://archive.ics.uci.edu/ml/index.php   (UCI)
https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/  (Libsvm) 


 The process of running the code is as follows:
 1) Input the data cotaining the filepath into the file "main.m" ;
 2) Then run the file "main.m";you will get the AUC,F1,G-M and Mcc values;
 3) The Subsets generation is in the file "demo_FCMZ.m";
 4) The MlFCM is contained in the file " multilayerFCM.m";
 5) The variable optimization in LGSCM is cotained in the file "HSIC_MC";
 6) The classifier C4.5 is contained in the file "Use_C4_5.m".
